# 📁 快速迁移到D盘 - 简化步骤

## 方法1：Windows资源管理器（最简单）⭐

### 步骤：
1. **打开文件夹**
   - 按 `Win + E` 打开资源管理器
   - 导航到：`C:\Users\黎又榜\每日新闻推送系统\deepread`

2. **复制文件夹**
   - 右键点击 `deepread` 文件夹
   - 选择"复制"

3. **粘贴到D盘**
   - 在左侧导航栏点击"此电脑" → "本地磁盘 (D:)"
   - 在D盘空白处右键 → "粘贴"
   - 等待复制完成（约10-30秒）

4. **验证**
   - 打开 `D:\deepread`
   - 确认文件完整（应该有很多 .py 文件）

5. **测试新位置**
   ```bash
   # 打开PowerShell（Win + X → Windows PowerShell）
   cd D:\deepread
   python launch.py
   ```

6. **删除旧文件**（可选）
   - 确认新位置正常后
   - 删除 `C:\Users\黎又榜\每日新闻推送系统\deepread`

---

## 方法2：使用PowerShell命令

1. **以管理员身份打开PowerShell**
   - 按 `Win + X`
   - 选择"Windows PowerShell (管理员)"

2. **执行复制命令**
   ```powershell
   Copy-Item -Path "C:\Users\黎又榜\每日新闻推送系统\deepread" -Destination "D:\" -Recurse -Force
   ```

3. **测试**
   ```powershell
   cd D:\deepread
   python launch.py
   ```

---

## 迁移后启动方式

**命令行**：
```bash
cd D:\deepread
python launch.py
```

**创建快捷方式**：
1. 右键 `start_v3.8.bat` → 发送到 → 桌面快捷方式
2. 右键快捷方式 → 属性
3. 修改"起始位置"为：`D:\deepread`
4. 点击"确定"
