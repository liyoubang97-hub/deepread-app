- **改进**：《原子习惯》等可用书籍居中占满一行
- **效果**：更突出，更易点击

### 4. ⬜ 移除所有有色框（极简设计）
- **核心观点框**：米色渐变 → 白色+黑色左边框
- **内容卡片**：淡灰色背景 → 白色+灰色左边框

- **金句框**：淡灰色背景 → 白色+灰色左边框
- **效果**：极简优雅，突出内容

### 5. 📍 修复跳转后滚动到顶部
- **问题**：跳转后停留在底部
- **解决**：使用多种方法确保滚动
- **代码**：
  ```javascript
  window.scrollTo({ top: 0, behavior: 'smooth' });
  document.documentElement.scrollTop = 0;
  document.body.scrollTop = 0;
  ```

## 💡 设计理念

### 极简主义
- **移除所有装饰性背景色**
- **只保留必要的左边框**
- **让内容成为焦点**

### 视觉层次
- **核心观点**：4px黑色左边框
- **重点内容**：3px深灰左边框
- **普通内容**：3px浅灰左边框

### 色彩方案
- **主色**：#2D3436（深灰）
- **辅助色**：#636E72（中灰）
- **边框色**：#E8EEF2（浅灰）
- **背景色**：#FFFFFF（纯白）

### 两列布局实现

```python
for i in range(0, len(BOOKS_DATA), 2):
    book1 = BOOKS_DATA[i]
    book2 = BOOKS_DATA[i + 1] if i + 1 < len(BOOKS_DATA) else None

    if book2:
        col1, col2 = st.columns(2)
        with col1:
            render_book_card(book1)
        with col2:
            render_book_card(book2)
```

### 滚动到顶部（多重保障）

```javascript
window.addEventListener('load', function() {
    setTimeout(function() {
        window.scrollTo({ top: 0, behavior: 'smooth' });
        document.documentElement.scrollTop = 0;
        document.body.scrollTop = 0;
    }, 150);
});
```
