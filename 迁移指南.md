## 🚀 手动迁移步骤（2分钟完成）

### 方法1：使用Windows资源管理器（推荐）⭐

1. **打开文件夹**
   - 导航到：`C:\Users\黎又榜\每日新闻推送系统\deepread`

2. **复制文件夹**
   - 右键点击 `deepread` 文件夹
   - 选择"复制"

3. **粘贴到D盘**
   - 打开"此电脑" → D盘
   - 在空白处右键 → "粘贴"
   - 等待复制完成（约1-2分钟）

4. **验证**
   - 确认 D:\deepread 文件夹已创建
   - 打开 D:\deepread 检查文件是否完整

5. **测试新位置**
   ```bash
   cd D:\deepread
   python launch.py
   ```

6. **删除旧文件**（可选）
   - 确认新位置正常后
   - 删除 C:\Users\黎又榜\每日新闻推送系统\deepread

