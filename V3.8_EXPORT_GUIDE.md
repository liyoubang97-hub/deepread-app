## 📤 两种导出模式

### 模式1：📝 导出我的笔记

**包含内容**：
- 只有用户填写的思考答案
- 不包含书籍内容
- 适合：已有书籍，只想保存自己的思考

**使用场景**：
- 你已经读过这本书，有纸质版
- 只想保存自己的思考笔记
- 文件更小，更简洁

---

### 模式2：📚 导出完整笔记

**包含内容**：
- 📖 引言（为什么要读）
- 💡 核心洞察（3个核心观点）
- ✅ 实践步骤（行动指南）
- 🤔 反思问题 + **你的答案**
- 📝 金句摘录

**使用场景**：
- 完整的学习资料
- 可以分享给他人
- 建立个人知识库
- 复习回顾

## 🔧 如何使用

### 步骤1：阅读并填写笔记

1. 选择一本书（如《思考，快与慢》）
2. 阅读引言、洞察、实践
3. 在"反思"页面填写思考题
4. 看到绿色✓表示已保存

### 步骤2：导出笔记

在"反思"页面底部，有两个按钮：

**按钮A：📝 导出我的笔记**
```
点击 → 生成"导出我的笔记"按钮
      → 点击"⬇️ 下载笔记文件"
      → 保存到本地
```

**按钮B：📚 导出完整笔记**
```
点击 → 生成"导出完整笔记"按钮
      → 点击"⬇️ 下载完整笔记"
      → 保存到本地
```

### 步骤3：导入飞书（或其他工具）

#### 方法1：飞书文档
1. 打开[飞书文档](https://feishu.cn)
2. 创建新文档
3. 点击"导入" → 选择"Markdown"
4. 选择下载的.md文件
5. 完成！

#### 方法2：Obsidian（本地知识库）
1. 打开Obsidian
2. 将.md文件放入你的vault文件夹
3. Obsidian会自动识别并索引
4. 可以添加标签、双链等

#### 方法3：Notion
1. 在Notion中创建页面
2. 复制.md文件内容
3. 粘贴到Notion页面
4. 调整格式（Notion会自动转换Markdown）

#### 方法4：其他支持Markdown的工具
- Typora
- 印象笔记
- 有道云笔记
- GitBook

## 📞 常见问题

### Q: 为什么使用Markdown格式？

A: Markdown是轻量级标记语言，优势：
- ✅ 纯文本，永不丢失
- ✅ 支持所有主流工具
- ✅ 可以版本控制（Git）
- ✅ 文件小，易于管理

### Q: 导出的文件可以在手机上查看吗？

A: 可以！方法：
1. 保存到云盘（百度网盘、阿里云盘等）
2. 手机安装Markdown查看器（如MWeb、Typora Mobile）
3. 或直接导入飞书手机版

### Q: 笔记会保存在DeepRead平台上吗？

A: 目前（V3.8）笔记保存在浏览器session中，刷新会丢失。
**建议**：每次填写完立即导出保存。

**未来**：V4.0将支持云端保存和账号系统。

### Q: 可以编辑导出的Markdown文件吗？

A: 完全可以！Markdown就是纯文本，可以用任何文本编辑器打开修改：
- Windows: 记事本、VSCode、Typora
- Mac: TextEdit、Typora、VSCode
- 手机：MWeb、iA Writer
