## 📖 新增书籍详情

### 1. 《刻意练习》
**作者**: 安德斯·艾利克森

**核心价值**: 揭秘天才的秘密 - 卓越不是天赋，而是正确的练习方法

**3个核心洞察**:
1. **天赋的陷阱** - 你以为的"一万小时"其实大部分是浪费
2. **走出舒适区** - 真正的学习发生在你感到吃力的地方
3. **即时反馈** - 没有反馈的练习，只是重复错误

**适合人群**: 想在任何领域达到顶尖水平的人

---

### 2. 《深度工作》
**作者**: 卡尔·纽波特

**核心价值**: 在分心的世界中，深度专注工作的能力是日益稀缺的资产

**3个核心洞察**:
1. **深度工作vs浮浅工作** - 你的时间被哪种工作占据了？
2. **专注力是稀缺资源** - 为什么高效人士越来越稀缺？
3. **拥抱无聊** - 训练大脑适应深度专注

**适合人群**: 知识工作者、创作者、想要提升效率的人

---

### 3. 《原则》
**作者**: 雷·达里欧

**核心价值**: 世界最大对冲基金创始人的人生智慧

**3个核心洞察**:
1. **拥抱现实** - 世界这台机器是如何运转的？
2. **五步流程** - 目标→问题→诊断→设计→执行
3. **极度开放** - 克服大脑的自我防御机制

**适合人群**: 管理者、创业者、想要系统化思考的人

## 🎯 内容结构说明

每本书都包含完整的5个模块：

### 📖 引言
- 为什么要读这本书？
- 能解决什么问题？
- 适合谁读？

### 💡 核心洞察（3个）
- 每个洞察包含：核心观点、为什么重要、现实案例、思考问题

### ✅ 实践步骤（3个）
- 具体可执行的行动指南
- 从理论到实践的转化

### 🤔 反思与思考（5个问题）
- 深度思考题
- 连接个人经验
- 支持用户填写并导出

### 📝 值得记住的话
- 精选金句摘录
- 便于复习回顾

## 🚀 立即体验

### 启动方式

```bash
# 方式1：双击启动
start_v3.8.bat

# 方式2：命令行
python launch.py

# 方式3：直接运行
python -m streamlit run app_v3.8.py
```

### 使用流程

1. 打开应用（浏览器自动打开）
2. 在首页看到5本书的卡片（新增3本标记为"可阅读"）
3. 点击任意一本书开始深度阅读
4. 完成引言→洞察→实践→反思
5. 填写反思问题
6. 导出笔记到飞书/Obsidian/Notion
